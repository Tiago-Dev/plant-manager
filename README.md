- React Native
- Typescript
- Expo
- Lottie React Native
- Styled Components


## Run app
- Change the variable baseURL in the src/services/api.ts file to your IP 

Ex: baseURL: 'http://{YOUR IP HERE}:3333',

- Run the following commands:
